.SH NAME

mica - Mutual Information Coevolution Analysis

.SH SYNOPSIS

.B mica [options]

.SH AVAILABILITY

All UNIX flavors

.SH DESCRIPTION

mica performs various published coevolution detection using mutual information on patterns.
