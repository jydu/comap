CoMap

A C++ shell program to detect co-evolving groups of sites

Download

Main

CoMap may be downloaded from http://download.gna.org/comap. The source code is available as a .tar.gz archive, and requires the Bio++ libraries Utils, NumCalc, SeqLib and PhylLib to be installed (see http://kimura.univ-montp2.fr/BioPP). A static executable is also available for the Linux platform.
